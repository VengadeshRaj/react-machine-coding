# React Machine Coding Topics

This repository includes different types of React machine coding challenges and their solutions. Each folder inside this repo contains a single machine coding topic and its solution.

Please go through the `README.md` of each folder before jumping into the solutions.

## Common Prerequisites:

- Node.js installed.

## Common Technologies Used:

- React JS
- Tailwind CSS

## List of Machine Coding Topics:

| Topic Name | Description |
|---|---|
| find-pair-card | A memory game based upon the numbers in pair of cards. |
| stopwatch | A stopwatch app, User can able to start, stop and reset the timer. |
| tic-tac-toe | Popular two player game, player has to mark the symbole(X or O) in order of straight or cross. |
| to-do-list | Users can add new to-do items and move them through the stages, To Do, In Progress, and Done |
| traffic-light | A similar implementation of a traffic light signal can be seen on roads |
| otp-login | A 4-digit OTP input component with auto-focus and seamless cursor navigation.|
| progress-bar | A progress bar loader automatically increments from 0% to 100%.|
| nested-comment-section | A Nested comment section, User can able to add,edit,up-vote,down-vote,delete comment and its replies.|
| searchable-text-box | A searchable text box that allows users to look up fruits, displaying dynamic suggestions based on their input used debouncing mechanism to minimize unnecessary event triggers.|
| linkedin-chat-section | Replica of the LinkedIn chat |
